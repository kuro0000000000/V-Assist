# V-Assist — Webカメラだけで動く VTuber アバター配信ツール

**V-Assist** は、Webカメラ1台で顔・体・手指をトラッキングし、**VRMアバター**を透過ウィンドウに表示する Windows 向けデスクトップアプリです。VSeeFace 等の外部トラッキングソフトや専用機器（Leap Motion 等）は不要で、**アプリ単体・完全オフライン**で動作します。透過ウィンドウなので OBS のウィンドウキャプチャでそのまま配信に使えます。

- 描画 / UI: **Electron + three.js + [@pixiv/three-vrm](https://github.com/pixiv/three-vrm) v3**（VRM0 / VRM1 両対応）
- トラッキング: **[MediaPipe Tasks Vision](https://developers.google.com/mediapipe)**（Face / Pose / Hand Landmarker）を同梱しオフライン実行
- サーバー不要・ネットワーク通信なし（モデル・wasm はローカル同梱、独自スキームで配信）

---

## 主な機能

### トラッキング（Webカメラ）
- **顔**: 頭の向き（首/頭に分配）、視線（黒目）、まばたき、口形（あいうえお）、表情
- **パーフェクトシンク対応**: ARKit 52ブレンドシェイプを持つ VRM には直接パススルーし、眉・頬・口の細かな表情まで再現。非対応モデルは従来のヒューリスティック変換に自動フォールバック
- **体・腕**: 上半身の傾き・腕の上げ下げ（OFF時は自然に下ろした姿勢）
- **手・指**: 手のひらの向き・手首のひねり・各指の曲げ（手ローカル基準で算出し、手の向きに左右されず忠実に追従）
- **マイク音声で口パク補助**（任意）: 映像の口形と音声を合成して口の反応を安定化
- **アイドル動作**: トラッキングOFF/ロスト時も、自動まばたき・呼吸・微小な頭の揺れで自然に見せる

### 配信向け演出
- **アクション表示**: 喜び ♪ / 怒り 💢 / 悲しみ 💧 / 驚き ❗ の装飾アイコン、キラキラ ✦ / ハート ♥ / バースト ✸ のエフェクトを、設定ボタンや**グローバルホットキー**から表示
- **配色テーマ**切り替え
- 透過背景でアバターのみを表示（OBS取り込み向け）

### ウィンドウ / 操作
- フレームレス透過ウィンドウ（ドラッグ移動・最前面表示・クリックスルー切替）
- **カメラプレビュー**サブウィンドウ（撮影状況の確認用）
- タスクトレイ常駐、操作バー右上の **× ボタン**で終了
- D&D または選択ダイアログで `.vrm` を読み込み、次回起動時に自動復元

## 動作環境

- **OS**: Windows 10（1809 / build 17763 以降）/ Windows 11（64bit）
- **Webカメラ**: 顔・体・手トラッキングに使用（720p以上推奨）
- **マイク**: 音声リップシンク利用時のみ（任意）
- **GPU**: WebGL 2.0 / DirectX 11 対応（内蔵GPU可。専用GPU推奨）
- **メモリ**: 4GB以上（8GB以上推奨。OBS併用なら16GB推奨）

> VRMモデルは [VRoid Studio](https://vroid.com/studio) で出力したモデルでの動作を確認しています。

## 使い方

1. アプリを起動し、`.vrm` ファイルをウィンドウへドラッグ&ドロップ（または「ファイルを選択」）。
2. ウィンドウにマウスを乗せると操作バーが表示されます。📷 ボタンで **Webカメラトラッキングを開始**。
3. 🎯 ボタンで正面を向いた状態を**較正**（ニュートラル姿勢の取得）。
4. OBS の「ウィンドウキャプチャ」で V-Assist ウィンドウを取り込むと、透過背景でアバターを配信できます。

設定ウィンドウ（⚙ ボタン）で、使用カメラ・鏡像・スムージング・各種感度（頭/まばたき/口/眉/視線）・体や腕トラッキングの有効化・音声リップシンク・カメラプレビュー表示・テーマ・アクション表示・ホットキーなどを調整できます。

## ライセンス形態（試用版 / Pro）

Microsoft Store での「**無料アプリ + アプリ内購入（Pro解除）**」配布を想定しています。

- **試用版（無料）**: 全機能を利用可能。ただし表示に「V-Assist 試用版」ウォーターマークが表示されます。
- **Pro**: アプリ内購入でウォーターマークを除去。
